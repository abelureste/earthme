# Earthme

A WIP miniature social media website to display and share where you've been in the world.

![Cipherme](https://github.com/abelureste/earthme/blob/main/src/assets/earthme-demo.gif)

## Features

The core features:

- A 3D earth with manipulable view
- Accurate and toggleable meridian lines
- Ability to add pins to earth using coordinates
- Basic user profile information

Basic social media features:

- Create/share user profiles
- Search for user profiles
- Interact with user's pin list
